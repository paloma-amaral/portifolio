## 🌟 Sobre o Projeto

Este repositório contém o código-fonte do meu portfólio pessoal. A aplicação foi desenhada para refletir meu perfil "híbrido", focado em resolver problemas operacionais através da tecnologia. 

O site é altamente responsivo, acessível e conta com um sistema dinâmico de troca de perfis (CLT, Consultoria PJ, Dados), alterando a comunicação visual e o copywriting de acordo com a necessidade do visitante.

🔗 **[Acesse o Portfólio ao vivo aqui](https://paloma-amaral.vercel.app)** *(Substitua pelo link da Vercel depois)*

## 🛠️ Tecnologias Utilizadas

- **[Next.js](https://nextjs.org/)** (App Router) — Framework React focado em performance.
- **[React](https://react.dev/)** — Biblioteca para construção de interfaces de usuário.
- **[Tailwind CSS](https://tailwindcss.com/)** — Framework CSS utilitário para estilização rápida e responsiva.
- **[Framer Motion](https://www.framer.com/motion/)** — Biblioteca para animações fluidas e microinterações.
- **[TypeScript](https://www.typescriptlang.org/)** — Adicionando tipagem estática ao JavaScript.

## ⚙️ Como rodar localmente

Caso queira explorar o código na sua própria máquina:

1. Clone este repositório:
   ```bash
   git clone https://github.com/palomaamaral/portifolio.git
   ```
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Rode o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
4. Acesse `http://localhost:3000` no seu navegador.
